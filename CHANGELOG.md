# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- WebSocket server implementation using Ratchet
- JWT-based authentication
- Room/Channel management
- Connection management
- Event system
- Middleware support
- Presence system (online/offline tracking)
- Rate limiting
- Message history (optional)
- IP-based connection limiting
- SSL/TLS support
- CORS configuration
- Logging system
- Helper functions
- CLI commands (start, publish)
- Comprehensive documentation

### Features
- Modular architecture
- Easy to extend
- Production ready
- Well documented
- Type-safe code
- PSR-4 autoloading
